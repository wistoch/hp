Playing Sounds

Once one or more AudioBuffers are loaded then we're ready to play sounds. Let's assume we've just loaded an AudioBuffer with the sound of a dog barking and that the loading has finished.

Then we can play this dogBarkingBuffer with a the following code. 

var context = new webkitAudioContext();

function playSound() {
    var source = context.createBufferSource();  // creates a sound source
    source.buffer = dogBarkingBuffer;           // tell the source which sound to play
    source.connect(context.destination);        // connect the source to the context's destination (the speakers)
    source.noteOn(0);                           // play the source now
}

This playSound() function could be called every time somebody presses a key or clicks something with the mouse.

Even better, we could make the function take the buffer and time as arguments: 

var context = new webkitAudioContext();

function playSound(buffer, time) {
    var source = context.createBufferSource();
    source.buffer = buffer;
    source.connect(context.destination);
    source.noteOn(time);
}

Then a different sound could be played depending on different events (like which key is pressed). Normally, a time of 0 would be used to "play sound now". But, sounds can be played at exact times to create rhythms.
