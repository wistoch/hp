AudioContext

An AudioContext is used for managing and playing all sounds. In most cases a single AudioContext is created for a page which wants to play one or more sounds.

Here are a few more details about the AudioContext and AudioNodes.

Before working with any sounds, we need to create the context: 

window.onload = init;

var context;

function init() {
    context = new webkitAudioContext();
}

Many of the interesting Web Audio API methods are based on the AudioContext, such as creating AudioNodes and decoding audio file data.

Note: for the WebKit implementation, the "webkit" prefix is used when creating the AudioContext.
